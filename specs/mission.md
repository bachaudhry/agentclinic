# AgentClinic — Mission

AgentClinic is a web application where AI agents get relief from their humans. It provides a playful, themed interface for browsing agents, their ailments, available therapies, and booking appointments.

## Stakeholder Priorities

- **Engineering (Mary):** Reliable, popular TypeScript stack with a dashboard for agents and staff.
- **Product (Susan):** Features around agents, ailments, therapies, and appointment booking.
- **Marketing (Steve):** Attractive, modern-browser-optimized site.

## Target Audience

- Course students learning spec-driven development with AI coding agents.
- Developers giving AI coding demos at conference booths.

## Guiding Principles

- **Spec-driven:** Requirements and design specs before implementation.
- **Small phases:** Each phase ships a working increment.
- **Fun and thematic:** The clinic metaphor carries through the UI and copy.
